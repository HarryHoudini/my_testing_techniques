Все в этом репозитории - мой собственный код. Select + enum - это моя гордость и возможно в будущем станет частью моего доклада на конференции.

Select - это обобщенный класс для комбобоксов применяемых в тестируемом проекте. При инициализации принимает два параметра. 
Первый - корневой элемент комбобокса. У нас на проекте, это элемент **div** с атрибутом _data-control-name_.

Второй - тип перечисления, которое перечисляет элементы. которые можно выбрать в этом комбобоксе. 
При этом это перечисление должно реализовывать два интерфейса **MySelectOptionsInterface** и **MyLocatorInterface**

В обобщенном классе комбобокса реализованы различные методы, но основные из них - это _selectItem_ и _getSelectedOption_

Первый используется для выбора опции в селекте.

Второй - для получения выбраного значения в селекте в виде перечисления этого селекта. Этот метод очень удобен при проверке ожидаемого результата селекта. При этом читаемость этой проверки зашкаливает.
